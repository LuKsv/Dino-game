# Dino-game
Projekt na mikrokontroler ESP32, który jest grą w tkz DINO ,znaną z przeglądarki google. Projekt jest prostą grą sterowaną przyciskiem!
NARZĘDZIA:
-mikrokontroler ESP32
-płytka stykowa
-kable męsko męskie, oraz męsko żeńskie
-Wyświetlacz LCD
-Konwenter L2C
Jest to projekt zrobiony jak wszystkie moje wrzucone na tą stronę z zajawki do elektroniki, jest to prosta gra zaprogramowana w arduinoIDE, w której można za pomocą przycisku zagrać w kultową dla naszego pokolenia gre:)
